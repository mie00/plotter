# Plotter

A java library to plot mathemetical expressions with variables.

## Usage

### Requirements

1. jdk 1.8.0\_45
2. javafx 8.0.45

I think they can run on previous versions, but I haven't tried.

### Download and Installation

```bash
git clone https://www.github.com/mie00/plotter
cd plotter
make
```

### Running

`make run`
